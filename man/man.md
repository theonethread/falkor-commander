# NAME

**falkor-commander** - Excerpt

# SYNOPSIS

```
falkor-commander [(-- <answers>...)]
falkor-commander [(-- <answers>...)]
falkor-commander (-v | --version | -h | --help)
```

# DESCRIPTION

Description

# OPTIONS

`-v`, `--version`
:   Show version and exit

`-h`, `--help`
:   Show help and exit

`-- <answers>...`
:   Treat all positional arguments after double dash as buffered answers
